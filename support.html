Contact route

If the FAQ does not solve it, use the contact page so the message reaches the support inbox.

Quick fixes

  • Confirm notification access is enabled for Notisaver.
  • Disable battery optimization for reliable background capture.
  • Verify the source app is not blocked inside per-app controls.
  • Restart the listener service after Android system updates.

Include these details

  • Android version and device model
  • App version or APK filename
  • Short problem description and expected behavior
  • Whether the issue affects capture, export, backup, or app lock
